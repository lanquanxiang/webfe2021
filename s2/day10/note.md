浮动和定位（改变元素的位置）
# 1. 文档流
# 2. 盒子模型
1. 构成：元素、内边距、边框、外边距
2. 装饰顺序：上右下左（顺时针）
3. 值的复制
# 3. 浮动
1. 脱离文档流【掌握】
2. float:left;right （导航栏）【掌握】
3. 选修：浮动的元素（上一块级元素之后）
扩展：清除浮动带来的影响  clear:both（清除浮动）不允许浮动元素在该元素的左右两侧
# 4. 定位 position
1. position: static;         /*默认原位置，无特殊效果*/
2. position: relative;     /*不脱离文档流，默认原始位置，相对于原位置*/
3. position: absolute; (子绝父相)    /*脱离文档流，默认原始位置，相对于有特殊定位的父元素*/
4. position: fixed;        /*脱离文档流，默认原始位置，相对于窗口*/
# 5. dispaly 显示
1. block 让元素以块元素的形式显示
2. inline 让元素以行内元素的形式显示
3. inline-block 让元素以行内块元素的形式显示
4. none 让元素消失（从文档流中消失）
