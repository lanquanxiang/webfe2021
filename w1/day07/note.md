CSS
# 1. 概述
1. 是什么：层叠样式表（网页装饰）
2. 为什么：将网页的内容（HTML）以及样式（CSS）进行分离
> 易维护
> 易扩展
> 易移植
降低成本
# 2. CSS语法
## 1. 构成
```
选择器{
	声明1；
	声明2；
	……
	}
```
> 样式表中可以同时进行多条声明，建议一行写一条
> 声明 （属性:属性值;）
> /*注释*/   （//错误）
## 2. 在网页中的使用(掌握)
> 1. 行内样式
在标签中添加style属性，在属性值中进行样式声明
> 2. 内部样式
在head添加style标签，然后再标签体中进行样式声明
> 3. 外部样式
将样式表保存到*.css文件中，在链接到页面中
