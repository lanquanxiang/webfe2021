# 2. 定位position
1. position: static;         /*默认原位置，无特殊效果*/
2. position: relative;     /*不脱离文档流，默认原始位置，相对于原位置*/
3. position: absolute;(子绝父相)/*脱离文档流，默认原始位置，相对于有特殊定位的父元素*/
4. position: fixed;        /*脱离文档流，默认原始位置，相对于窗口*/
