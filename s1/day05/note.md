# 2. 下拉列表
1. 单选、多选选项太多->下拉列表
2. 语法 select>option
3. 属性
> selected 默认选中
> size 一次可见的选项数（默认是1）
> multiple 可以多选
# 3. 文本区域标记
1. 文本框（单行）->内容太对->文本区域
2. 语法： textarea
3. 属性：
> 默认值写在元素的文本中
> rows 行
> cols 列（字符）
扩展知识：
1. 如果需要给表单加上框和备注，form>fieldset>legend
2. ```<label for="username"> ```扩大获得焦点的范围
